## ✨ ویژگی‌ها

### 📱 **PWA قوی**
- ✅ نصب روی موبایل و دسکتاپ
- ✅ کار آفلاین کامل
- ✅ UI ریسپانسیو و مدرن
- ✅ Service Worker برای کش کردن

### 👥 **مدیریت مشتریان**
- 📝 ثبت نام و شماره مشتری
- 🏷️ تولید ID خودکار ۸ رقمی
- 🔍 جستجوی پیشرفته (نام، شماره، مدل، قیمت)
- 📊 آمار زنده (تعداد مشتریان، سفارشات)

### 📏 **سیستم اندازه‌گیری**
- 🧮 ۱۷ فیلد اندازه‌گیری استاندارد
- ⚡ ذخیره خودکار (Auto-save)
- ⌨️ ناوبری با کیبورد (Tab, Ctrl+S)
- 📱 ورودی آسان در موبایل

### 🎨 **سیستم مدل‌ها**
- 🧥 ۷ مدل یخن (آف دار، چپه، پاکستانی، ...)
- 👕 ۶ مدل آستین (کفک، ساده، بندک، ...)
- 👗 ۵ مدل دامن (یک بخیه، دوبخیه، چهارکنج، ...)
- 🔧 ۵ ویژگی اضافه (جیب رو، جیب شلوار، ...)

### 💰 **قیمت و تحویل**
- 💵 قیمت به افغانی
- ✅ تیک رسید پرداخت
- 📅 انتخاب روز تحویل (شنبه تا جمعه)
- 🏷️ چاپ لیبل حرفه‌ای

### 🎨 **تم‌های متعدد**
- 🌙 **حالت تاریک** (پیش‌فرض)
- ☀️ **حالت روشن**
- 🌈 **حالت ویوید** (رنگی)
